# Contributing to Suchify Theme Template

Thank you for your interest in improving the Suchify Theme Template! This document provides guidelines for contributing.

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion:

1. Check if the issue already exists
2. Create a detailed issue report including:
   - Description of the problem
   - Steps to reproduce
   - Expected behavior
   - Actual behavior
   - Browser/OS information

### Suggesting Improvements

We welcome suggestions for:
- New features
- Better documentation
- Code improvements
- UI/UX enhancements

### Code Contributions

1. **Fork the repository** (if applicable)
2. **Create a feature branch**
3. **Make your changes**
4. **Test thoroughly**
5. **Submit for review**

## Development Guidelines

### Code Style

- Use consistent indentation (2 spaces)
- Write clear, commented code
- Follow existing patterns
- Keep functions focused and small

### Testing

- Test all changes locally
- Ensure no console errors
- Test in multiple browsers
- Verify responsive design

### Documentation

- Update README if needed
- Add comments for complex logic
- Document new features

## Questions?

Contact the Suchify development team:
- **Email**: dev-support@suchify.com
- **Partner Support**: partners@suchify.com
